2.2 CLAUDE.md 核心内容规范

# StudyHere — AI 上下文文件

## 项目简介
AI 学习闭环平台。用户上传资料+设定目标 → AI生成学习计划 → 学练考纠全闭环。

## 技术栈
- Frontend: Next.js 14 App Router + TypeScript + Tailwind + shadcn/ui
- Backend: Hono.js + Prisma + PostgreSQL (Supabase)
- AI: OpenAI GPT-4o / GPT-4o-mini (见 apps/api/src/lib/openai.ts)
- Queue: BullMQ + Upstash Redis

## 代码规范
- 所有文件使用 TypeScript，禁止 any
- API 响应统一格式: { success: boolean, data?: T, error?: string }
- 错误处理: 使用 Result 类型，不 throw 到业务层
- 组件命名: PascalCase；函数/变量: camelCase

## 共享类型位置
所有 API 类型定义在 packages/types/src/ 下，前后端复用。

## 常用命令
pnpm dev          # 启动所有服务
pnpm build        # 全量构建
pnpm test         # 运行测试
pnpm db:migrate   # 执行数据库迁移

5.3 Prompt 工程规范

规范项要求示例

Few-shot 示例每个 Prompt 包含 2-3 个高质量示例在 docs/prompts/ 中每个文件标注示例
版本管理Prompt 文件命名 功能-v版本号.txt，变更记录 changelogextract-v2.txt

Token 预算每个 Prompt 标注预估 input/output tokens避免超限，控制成本
